Actividad Práctica – Registro de Estudiantes

Manipulación de cadenas y uso del objeto Math en JavaScript.

Instrucciones

Imagina que desarrollas una función para gestionar un sistema de registro de estudiantes. Completa los siguientes requerimientos usando métodos de cadenas y el objeto Math:

  1. Crea una variable nombreCompleto con el nombre de un estudiante (p. ej., "Carlos Perez Gomez").
  2. Muestra cuántos caracteres tiene nombreCompleto.
  3. Verifica si el nombre incluye el apellido "Perez".
  4. Crea una segunda variable con "Sofia Gutierrez" y únelas (simula dos registros en una sola cadena).
  5. Reemplaza "Carlos" por "Juan" en nombreCompleto.
  6. Divide nombreCompleto en tres partes (nombre, primer apellido, segundo apellido) y muéstralas por separado.
  7. Convierte nombreCompleto a mayúsculas.
  8. Elimina espacios innecesarios al inicio y final de una cadena con espacios extra.
  9. Genera una calificación aleatoria entre 0 y 100 con Math.random() y redondéala con Math.round().

Código de la solución (solo lectura)


// 1) Variable con nombre completo
let nombreCompleto = "Carlos Perez Gomez";

// 2) Contar caracteres
let cantidad = nombreCompleto.length;

// 3) Verificar apellido
let tienePerez = nombreCompleto.includes("Perez");

// 4) Concatenar registros
let otroNombre = "Sofia Gutierrez";
let registros = nombreCompleto.concat(" | ", otroNombre);

// 5) Reemplazar "Carlos" por "Juan"
let reemplazado = nombreCompleto.replace("Carlos", "Juan");

// 6) Dividir en partes
let partes = nombreCompleto.split(" "); // ["Carlos","Perez","Gomez"]
let nombre = partes[0];
let apellido1 = partes[1];
let apellido2 = partes[2];

// 7) Mayúsculas
let mayus = nombreCompleto.toUpperCase();

// 8) Quitar espacios extra
let crudo = "   Carlos Perez Gomez   ";
let limpio = crudo.trim();

// 9) Calificación aleatoria (0–100, entero)
let calificacion = Math.round(Math.random() * 100);

// Salida formateada:
let salida = "";
salida += "Nombre Completo: " + nombreCompleto + "\\n";
salida += "Cantidad de caracteres: " + cantidad + "\\n";
salida += "Incluye 'Perez'?: " + tienePerez + "\\n";
salida += "Registros unidos: " + registros + "\\n";
salida += "Nombre reemplazado: " + reemplazado + "\\n";
salida += "Nombre: " + nombre + "\\n";
salida += "Primer apellido: " + apellido1 + "\\n";
salida += "Segundo apellido: " + apellido2 + "\\n";
salida += "En mayúsculas: " + mayus + "\\n";
salida += "Sin espacios extra: '" + limpio + "'\\n";
salida += "Calificación generada: " + calificacion + "\\n";

document.getElementById("out").textContent = salida;
Tip para estudiantes

Prueba cambiar los nombres, usar toLowerCase() o dividir por otro separador en split(). ¡Explora!
